class Client extends HttpAdapterComponent with Logging

A keen.io API client. A plain Client instance doesn't do a whole lotâ€”you'll want to mix in AccessLevels like Writer, Reader, or Master depending on the API operations that you need to perform.

The client defaults to using a built-in adapter for the Spray HTTP toolkit for HTTP requests. Another HTTP client library can be plugged in by implementing the HttpAdapter interface.
